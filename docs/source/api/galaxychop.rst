GalaxyChop API
==============

The modules are:

- **Core**: creates the galaxy to decompose.

- **Models**: performs dynamic decomposition using the Abadi method.

- **Sklearn_models**: performs dynamic decomposition using the Kmeans method.

- **Utils**: stuff
